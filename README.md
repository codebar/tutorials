## Getting started

This is a [GitHub Pages](https://pages.github.com/) repo, so you can render the pages with [Jekyll](http://jekyllrb.com/):
1. `gem install jekyll` (`sudo` if needed)
2. `jekyll serve`
3. go to http://127.0.0.1:4000
